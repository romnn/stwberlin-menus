============
Installation
============


Stable release
--------------

To install stwberlin menus, run this command in your terminal::

    $ pip install stwberlin_menus

If you don't have `pip`_ installed, see this `guide`_.

.. _pip: https://pip.pypa.io
.. _guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

To install stwberlin menus from the sources, first clone the repository::

    $ git clone git://github.com/romnnn/stwberlin_menus

After you obtained a local clone of the source, install with::

    $ python setup.py install
